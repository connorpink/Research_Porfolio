## Overview

Tile Slide Puzzle is an interactive puzzle project focused on both playability and algorithmic visibility.

## Solvers

The project includes multiple automated solving strategies:

- Greedy
- BFS
- A\*
- Weighted A\*
- Beam Search
- IDA\*

## Purpose

Rather than hiding the logic behind a simple game interface, the project exposes algorithmic tradeoffs and makes performance comparison part of the product itself.
